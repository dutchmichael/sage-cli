# Usage
<!-- usage -->
```sh-session
$ npm install -g sage-cli
$ sage COMMAND
running command...
$ sage (-v|--version|version)
sage-cli/1.0.0-alpha.1 linux-x64 node-v10.15.3
$ sage --help [COMMAND]
USAGE
  $ sage COMMAND
...
```
